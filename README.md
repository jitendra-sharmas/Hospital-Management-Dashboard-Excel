# 🏥 Hospital Management Dashboard (Microsoft Excel)

An interactive **Hospital Management Dashboard** developed using **Microsoft Excel**. This project demonstrates how Excel can be used for data analysis, visualization, and reporting through **Pivot Tables, Pivot Charts, KPI Cards, and Slicers**.

---

## 📌 Project Overview

The dashboard provides a clear overview of hospital data and helps analyze patient records, billing, diseases, and doctor performance. It is designed to make data easy to understand through interactive charts and filters.

---

## ✨ Features

- 📊 Total Patients KPI
- 💰 Total Bill KPI
- 🏥 Average Stay Days KPI
- 💵 Average Bill KPI
- 👨‍⚕️ Patients by Doctor
- 🦠 Patients by Disease
- 👨‍👩‍👧 Gender Distribution
- 💳 Bill by Doctor
- 🎛 Interactive Slicers for filtering data
- 📈 Dynamic Pivot Charts

---

## 🛠 Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Dashboard Design
- Data Visualization

## 📂 Dataset

The dataset contains hospital records including:
